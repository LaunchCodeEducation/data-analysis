15.3. Studio: EDA With Python Part 2¶

15.3.1. Getting Started¶

For this weeks studio fork this GitHub repository and clone to your computer.

If you need a refresher on how to do this see Instruction for Using Github w/Jupyter Notebooks.

15.3.2. Code Along¶

This studio follows the univariate EDA practice you did for this lesson’s reading. For the studio, use the IMDb popular movies from 2006 to 2016 dataset as you work through the notebook.

15.3.3. Submitting Your Work¶

When finished make sure to push your changes up to GitHub.

Copy the link to your GitHub repository and paste it into the submission box in Canvas for Studio: EDA Part 2 and click Submit.
