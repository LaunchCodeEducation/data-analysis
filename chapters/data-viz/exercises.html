18.2. Exercises: Data Visualization with Python¶

18.2.1. Getting Started¶

Fork this GitHub repository and clone to your computer. If you need a refresher on how to do this see Instruction for Using GitHub w/Jupyter Notebooks.

Read this article and work along using the GitHub repository you’ve just cloned to your computer.

18.2.2. Submitting Your Work¶

When finished make sure to push your changes up to GitHub. Copy the link to your GitHub repository and paste it into the submission box in Canvas for Exercises: Data Visualization w/Python and click Submit.
