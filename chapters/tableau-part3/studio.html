27.3. Studio: Tableau Part 3¶

27.3.1. Getting Started¶

27.3.1.1. Business Issue:¶

You work for TarMart corporate. TarMart has 45 department stores across the country, of various sizes. Between 2010 and 2013, TarMart has tracked weekly sales of each store by department. Your supervisors want some insights on which stores and departments are performing the best in weekly sales.

27.3.1.2. Setting Up the Studio:¶

Download Retail Data Analytics data set. There are 3 CSV files. To download all CSV files at once, click on the Download bubble next to the New Notebook bubble. An orange arrow is pointing to the Download bubble in the image below.

  1. Open the CSV file in Tableau Public.

  2. When you open this data set, you will want to create a relationship between the 3 CSV files.

    1. Each CSV has a Store column.

    2. Here’s some more context for working with relationships in Tableau:

      1. Relationships vs Joins.

      2. Don’t Be Scared of Relationships.

Note

This dataset has a large amount of rows that Tableau Public might have trouble querying at times. To mitigate this, try using only aggregated measures when dropping Weekly Sale into a sheet.

27.3.1.3. EDA¶

  1. What do weekly sales over time look like?

  2. Weekly sales by store type?

  3. Weekly sales by store?

  4. Weekly sales by department?

  5. How do sales by store and department change by quarter?

27.3.1.4. Calculations¶

  1. Determine a cutoff for constituting a high sale amount.

    1. Use your discretion from what you observe in your vizzes.

    2. Use a basic calculation to create a new field indicating your measure for high sales.

      1. Use this field in a new chart to display which stores return higher sales than others.

  2. TarMart wants to know which of the stores included in this dataset have contributed the most to overall sales in the company.

    1. Create a viz that uses the Quick Table Calculation Percent of Total to accomplish this.

27.3.1.5. Parameters¶

  1. Use a filter and parameter to change the number of stores displayed in the viz you created for Calculations Question 2.

    1. Make a copy of your viz by right clicking on the sheet tab and selecting Duplicate.

    2. Add the Stores pill to the filter tab in your new viz. Create a Top filter from your filter type choices. Note that Stores may initially be pulled in as a continuous measure due to its data type. Change it to a discrete dimension.

    3. Create a parameter that shows the Top Stores.

      1. Make the data type an integer.

      2. Make the allowable values a range.

      3. Set the range to a fixed, setting values from the Store column.

    4. Return to your Store filter and edit the Top options to use the Top Store parameter you have created.

    5. Add your filter parameter to the sheet.

      1. Right click on the Top Store parameter from the left hand Data pane, select Show Parameter.

  2. Return to your viz that displays the average weekly sales by department to use a parameter to display the top department marks in a distinct color.

    1. Right click on the Dept field to convert it to a dimension.

    2. Create a set of top departments using the Top Stores parameter.

    3. Drag the department set to the color option in the marks pane.

    4. Right click on the Top Stores parameter to show the parameter control in the viz.

    5. Rename the Top Store parameter to account for the fact that it’s now being used for both store and department dimensions.

27.3.1.6. Dashboard and Finesse¶

  1. Present your charts in a dashboard that summarizes your findings.

  2. Now is a good time to update labels and colors for an easily digestible report.

27.3.1.7. Finishing Touches¶

Before you turn in your vizzes:

  1. Make sure that they are easy to read.

  2. Review and edit any axes so that they don’t contain any file information.

  3. Make sure any filtering, group, or set information is easy to understand.

    1. For example, when using a set the predefined labels may say “In” and “Out”. Would extra context make them easier to understand?

    2. Don’t forget to title your charts.

  4. If you want to explore fonts and colors, go right ahead.

    1. Feel free to change the colors of any/all of your charts.

27.3.2. Submitting Your Work¶

When finished make sure to save and publish your work to your Tableau Public account. Copy the URL to your published Tableau project and paste it into the submission box in Canvas for Studio: Visualization with Tableau Part 3 and click Submit.
