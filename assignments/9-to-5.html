Assignment #3: 9 to 5¶

Your third graded assignment is due two weeks after the lesson 14, the fifth and final SQL lesson, and it is designed to be completed over several sessions. Do NOT put it off until the day before it is due!

Fork this GitHub repository. and clone it to your computer. If you need a refresher on how to do this see Instruction for Using GitHub w/Jupyter Notebooks.

Note

Instead of opening with Anaconda and Jupyter, open Azure Data Studio, click on file and then open, navigate to where you cloned the GitHub repository on your computer. Open the notebook 9to5.ipynb and follow the instructions to complete the assignment.

Submitting Your Work¶

When finished make sure to push your changes up to GitHub. Copy the link to your GitHub repository and paste it into the submission box in Canvas for Graded Assignment #3: 9 to 5 and click Submit.
