Checkpoint 5: Modeling Data¶

Before You Start¶

While working on this checkpoint, you will also have to present your work on the previous four checkpoints. We included notes on final project presentations at the end of this assignment. If you want to change anything about what you have done so far in earlier checkpoints, you do not have to re-submit any previous checkpoints unless your mentor requests you do so.

Getting Started¶

Fork this GitHub repository and clone it to your computer. Follow the instructions within the notebook to create a linear regression model with your cleaned dataset. If it doesn’t work out very well, that is alright. The most important thing is that you train a model and summarize the results.

Submitting Your Work¶
